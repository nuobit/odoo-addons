Installation
============

This module depends on :

- account_banking_sepa_direct_debit

This module is part of the NuoBiT/odoo-addons suite.

Configuration
=============

#. Go to Contacts > Configuration > Banks.
#. Search for a Banc Sabadell
#. Enable the field "Is Banc Sabadell"
